## ADDED Requirements

### Requirement: Gemma-backed image description
The system SHALL generate the final `describe_image` text using the LLM checkpoint `google/gemma-3-27b-it`.

#### Scenario: Successful description
- **WHEN** a task requests `type=describe_image`
- **THEN** the system returns a natural-language description generated by Gemma

### Requirement: Non-fatal fallback
If the Gemma model cannot be loaded or generation fails, the system SHALL fall back to returning the base caption from the vision captioning model.

#### Scenario: Gemma unavailable
- **WHEN** Gemma loading or generation raises an exception
- **THEN** the system returns the base caption instead of failing the request
